# Customer-Churn-Analysis Overview
This project focuses on analyzing customer churn behavior using Python. The goal is to identify key factors influencing customer churn and provide business insights to help reduce customer attrition.
The analysis is performed using a real-world telecom customer dataset with demographic, service usage, and billing information.


## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook


## Exploratory Data Analysis (EDA)

### 1. Churn Distribution
Customers who churned vs retained.

### 2. Churn by Gender
No major difference in churn rate between male and female customers.

### 3. Contract Type vs Churn
Month-to-month customers show significantly higher churn.

### 4. Services vs Churn
Customers without Online Security, Tech Support, and Device Protection show higher churn.

### 5. Payment Method vs Churn
Electronic check users have the highest churn rate.


## Key Business Insights
- Month-to-month contracts are the biggest churn risk group.
- Electronic check payment method has highest churn.
- Customers lacking security & support services churn more.
- Longer tenure customers are more loyal.
- Higher monthly charges increase churn probability.


## Business Recommendations
- Promote yearly contracts with discounts.
- Encourage auto-payment methods.
- Bundle security & support services.
- Provide loyalty offers to high-bill customers.
